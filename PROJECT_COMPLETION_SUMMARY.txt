## Technology Stack
- Language: Python 3.7+
- GUI: Tkinter (built-in)
- Data Processing: Pandas, NumPy
- ML: Scikit-learn (Random Forest)
- CV: OpenCV, Pytesseract
- Model Persistence: Pickle
- Threading: Python threading module

## Database
- Source: Goodreads Books Dataset
- Total Books: 11,123
- Total Authors: 6,639
- Columns: 12 (ID, Title, Authors, Rating, ISBN, Publisher, etc.)
- Size: ~6.5 MB CSV

## ML Model
- Algorithm: Random Forest Classifier
- Trees: 100
- Max Depth: 10
- Features: 3 (rating, ratings_count, text_reviews_count)
- Train Accuracy: 99.99%
- Test Accuracy: 99.78%
- Target: High/Medium/Low Demand

# INSTALLATION SUMMARY
# ===========================================================================

## Quick Start (Windows)
```
cd "Smart Library Assistant"
run_app.bat
```

## Quick Start (macOS/Linux)
```
cd "Smart Library Assistant"
chmod +x run_app.sh
./run_app.sh
```

## Manual Installation
```
pip install -r requirements.txt
python app.py
```

## First-Time Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Install Tesseract OCR (see SETUP_GUIDE.txt)
3. Run test: `python test.py`
4. Launch app: `python app.py`

# HOW TO USE
# ===========================================================================

## Search Books
1. Go to "Search Books" tab
2. Enter title or author name
3. Click Search or press Enter
4. View results in the text area
5. Click "Predict Demand" for ML predictions

## Ask Questions
1. Go to "Ask Question" tab
2. Type a natural language question
3. Example: "What is the rating of The Great Gatsby?"
4. System answers using dataset information only

## Upload Image & OCR
1. Go to "Image Upload & OCR" tab
2. Click Browse to select book cover image
3. Click "Extract & Search"
4. See extracted text and matching books

## Predict Demand
1. Perform a book search
2. Click "Predict Demand" button
3. View predictions in ML tab
4. See High/Medium/Low demand with confidence
